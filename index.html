Tentang Saya

Aditya Prasetya Suryana

Saya adalah seorang konsultan IT dengan pengalaman di sektor kesehatan dan manufaktur. Dengan keahlian dibidang IT, saya berkomitmen untuk memberikan layanan terbaik kepada klien. Saat ini, saya juga fokus pada proyek dan pengembangan teknologi.

Saya memiliki latar belakang pendidikan di bidang IT dan telah terlibat dalam berbagai proyek, saya aktif dalam meningkatkan keterampilan dan pengetahuan.

Proyek

ERP (Enterprise Resource Planning)

Pengembangan sistem ERP untuk meningkatkan efisiensi manajemen sumber daya perusahaan.

SIMRS (Sistem Informasi Manajemen Rumah Sakit)

Implementasi Sistem Informasi Manajemen Rumah Sakit untuk pengelolaan informasi kesehatan.

RME (Rekam Medik Elektronik)

Pengembangan aplikasi RME untuk mendukung pengelolaan dan pemantauan proses operasional.

Bridging SATUSEHAT

Integrasi sistem SATUSEHAT dengan aplikasi kesehatan untuk mendukung Kementerian Kesehatan.

Bridging BPJS Kesehatan

Integrasi layanan BPJS Kesehatan, termasuk Vclaim dan Pcare, untuk meningkatkan akses layanan kesehatan.

Bridging Mobile JKN

Integrasi aplikasi Mobile JKN untuk mempermudah akses layanan kesehatan bagi peserta BPJS Kesehatan.

Bridging Antrian Online

Pengembangan sistem antrian online (Console Box RS) untuk meningkatkan pengalaman pasien di rumah sakit.

Bridging SIRS

Integrasi Sistem Informasi Rumah Sakit (SIRS) Kementerian Kesehatan untuk memperbaiki pengelolaan data kesehatan.

SMART MONITORING

Implementasi IOT pada sistem monitoring suhu ruang operasi untuk memastikan kondisi optimal.
